    Represents an individual route with a name, path, params, values, etc.

    In general, you should never need to instantiate a Route directly. Use the RouteFactory instead, or the Map.

    The signature must be function(array $server, \ArrayObject $matches) and must return a boolean: true to accept this Route match, or false to deny the match. Note that this allows a wide range of manipulations, and further allows the developer to modify the matched params as needed.
