Package: Aura\View

Provides a TwoStepView pattern implementation; this allows for an inner ("core") view and and outer ("layout") view as commonly used by web apps.

mixed  $inner_view= ''

The inner view to use. This may be one of three types:

  • (string) A template file name.

  • (callable) A callable to execute. It should take no parameters.

  • (array) An array where each element key is a .format name, and the corresponding element value is a string or a callable. This type is most useful when allowing for multiple views using the same data.
